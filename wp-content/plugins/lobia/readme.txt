در این بسته فونت هایی نظیر: شبنم، وزیر، ساحل، تنها، پرستو، صمیم، گندم و استعداد  قرار گرفته است.

Lobia Persian font package for Persian WordPress users
In this package, fonts such as: Shabnam, Vazir, Sahel, Tanha, Parasto, .Parastoo and .Gandom are included.
After installing and activating the plugin, fonts will be added to your website by default, and also the font of the dashboard will be changed to the beautiful dew font.

* The fonts of this package are automatically added to the Advanced Editor Tools plugin (previously TinyMCE Advanced).



How to use the font anywhere on the site We have defined classes ready for your convenience that you can use anywhere on your website

Class names
.Shabnam .Vazir .Sahel .Tanha .Parastoo .Samim .Gandom

How to use

class = "Shabnam"
class = "Vazir"

== Frequently Asked Questions ==

= چطور میتونم فونت کل سایت و تغییر بدم؟ =

شما میتونید در فایل قالب یا سی اس اس سفارشی سایتتون خیلی راحت با قرار دادن کدهایی به عنوان مثال

body{
    font-family:  Shabnam !important;
}

h1,h2,h3{
    font-family:  Vazir !important;
}

فونت های قالب خودتونو تغییر بدید

= آیا این افزونه یا افزونه های مشابه سایت و سنگین میکنه؟ =

این افزونه به صورت بهینه با کمترین کد و بدون هیچ تنظیمات پیچیده ای ارائه شده افزونه های مشابه با پنل های پیچیده و استفاده از منابعی که برای افزونه تغییر فونت عملا ناکارامد هستند باعث استفاده از بخشی از منابع سایت شما میشه ولی در بسته فونت فارسی لوبیا سعی شده فقط کد مربوط به تغییر فونت و بهترین فونت های وب قرار بگیره و از هیچ تنظیمات و پنل پیچیده یا ... خبری نیست! پس با خیال راحت استفاده کنید.

= لایسنس  (مجوز) فونت ها چیه؟ =
فونت های قرار داده شده در بسته فونت فارسی لوبیا فونت آزاد (تحت مجوز SIL Open Font License 1.1) و رایگان هستند و از نظر لایسنس (مجوز) هیچ مشکلی برای سایت شما پیش نمیاد :)
